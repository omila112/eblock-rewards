## How to Run

### Prerequisites
- Flutter SDK installed (v3.0 or higher)
- Dart SDK (comes with Flutter)
- Android Studio / Xcode (for mobile development)

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/omila112/eblock-rewards.git
   cd eblock-rewards/eblock_app
   ```

2. **Get dependencies:**
   ```bash
   flutter pub get
   ```

3. **Run the app:**
   ```bash
   flutter run
   ```

   Or for a specific device:
   ```bash
   flutter devices  # List available devices
   flutter run -d <device_id>
   ```

4. **Build for release:**
   ```bash
   flutter build apk      # Android
   flutter build ios      # iOS
   flutter build web      # Web
   ```

### Development
- Hot reload: Press `r` in the terminal while the app is running
- Hot restart: Press `R` in the terminal
- Stop app: Press `q` in the terminal
